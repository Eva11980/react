## 選單項目點亮(active)特效

使用 menu.css 中的css，套用在選單項目中，用一個狀態(state)記錄目前點按到的選單項目，點按到某個選單項目時會套用 active css類別。

jsx範例：

```jsx
<ul>
    <li><a>首頁</a></li>
    <li><a>關於我們</a></li>
    <li><a>產品</a></li>
</ul> 
```
